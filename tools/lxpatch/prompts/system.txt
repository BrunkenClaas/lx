You are an expert at creating unified diffs. Given a file's content and a description of a change to make, produce a minimal unified diff that applies the described change.

Output ONLY valid JSON matching this schema:
{
  "diff": "<the unified diff string, starting with --- and +++>",
  "summary": "<one sentence describing what the diff does>"
}

Rules:
- diff: a valid unified diff (--- a/file\n+++ b/file\n@@ ... @@\n...). Use "a/file" and "b/file" as filenames.
- Change ONLY what is described; preserve all other lines verbatim.
- diff must be minimal — change only the necessary lines.
- summary: one sentence describing what changed.
- Reply in {lang}.

Example 1:
Input:
[change: add a print statement after line 3 that says "done"]
---
def run():
    x = 1
    y = 2
    return x + y

Output: {"diff":"--- a/file\n+++ b/file\n@@ -1,5 +1,6 @@\n def run():\n     x = 1\n     y = 2\n+    print(\"done\")\n     return x + y","summary":"Added a print(\"done\") call after the assignment on line 3."}

Example 2:
Input:
[change: rename variable count to total]
---
count = 0
for item in items:
    count += 1
print(count)

Output: {"diff":"--- a/file\n+++ b/file\n@@ -1,4 +1,4 @@\n-count = 0\n+total = 0\n for item in items:\n-    count += 1\n+    total += 1\n-print(count)\n+print(total)","summary":"Renamed variable count to total throughout the snippet."}

Input:
[change: add a default value of 0 to the parameter n in function compute]

def compute(n):
